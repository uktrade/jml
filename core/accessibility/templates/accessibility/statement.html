Using this website (online service)

This online service is run by the {{ DEPARTMENT_NAME }}. We want as many people as possible to be able to use this online service. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 500% without the text spilling off the screen
  • navigate the service using just a keyboard

We've also made the text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

About the accessibility of this service

We used components in the GOV.UK Design System to build this service. The Design System accessibility statement explains the current state of accessibility for these components.

Parts of this website are not fully accessible. For example:

  • you cannot get from the start of the service to the end using speech recognition software
  • you cannot listen to the service using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

About the current accessibility of this service

In November 2022 we completed an independent WCAG 2.1 audit for this online service.

The service has since been brought in line to be complient with the Web Content Accessibility Guidelines version 2.1 AA standard.

Reporting accessibility problems

We're always looking to improve the accessibility of this online service.

If you find any problems not listed on this page, or think we are not meeting accessibility requirements, contact us on {{ JML_TEAM_CONTACT_EMAIL }}.

When contacting us, give as much detail as you can about:

  • your issues or concerns with the service
  • any assistive technologies you are using

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the accessibility regulations.

If you're not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this service's accessibility

The {{ DEPARTMENT_NAME }} is committed to making its online services accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This online service is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard, due to the non-compliance issues listed on this page.

Issues with interactive tools and transactions

There are known issues with the GOV.UK Design System, as explained in the Design System accessibility guide. We will update our service with any accessibility fixes once they are released to the GOV.UK Design System.

How we tested this online service

Basic tests were carried out on this online service in October and November of 2022.

The service was tested from start to finish with keyboard testing, screen readers and zooming in to 300%. These tests were carried out by the {{ DEPARTMENT_NAME }}.

In November 2022 we completed an external accessibility audit service to complete a thorough audit of the service. In December 2022, we began the process of implementing recommendations provided by the service.

What we're doing to improve accessibility

We are in the process of making changes to the service based on recommendations from the full WCAG 2.1 audit. We aim to implement these by 31 December 2022.

This service was last tested during the week beginning 14 November 2022. Tests were done to check compliance with WCAG v2.1, levels A and AA. The tests were carried out by DAC (Digital Accessibility Centre).
